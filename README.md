# LazyListStudy

App de lista de tarefas usando Jetpack Compose.

## Tecnologias
- Kotlin
- Jetpack Compose
- Room
- Coroutines

## Funcionalidades
- Adicionar e remover tarefas
- Lista com LazyColumn
- Salvamento local com Room

Projeto criado para estudar listas com LazyColumn no Jetpack Compose.
